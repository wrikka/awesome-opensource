# Package Managers

| Reference | CLI | Description | Benefit |
|-----------|-----|-------------|---------|
| [bun](https://bun.sh/) | `bun` | JavaScript runtime | Fast JavaScript runtime as Node.js alternative |
| [fnm](https://github.com/Schniz/fnm) | `fnm` | Fast Node manager | Fast Node.js version management |
| [pnpm](https://pnpm.io/) | `pnpm` | Fast package manager | Fast and space-efficient package manager |
| [mise](https://mise.jdx.dev/) | `mise` | Dev tools manager | Manage multiple dev tools in one place |
| [cargo-binstall](https://github.com/cargo-bins/cargo-binstall) | `cargo-binstall` | Cargo binary installer | Fast Rust binary installation without compiling |
| [uv](https://github.com/astral-sh/uv) | `uv` | Python package manager | Fast Python package manager |
| [winget](https://learn.microsoft.com/windows/package-manager/) | `winget` | Windows package manager | Windows package manager |
