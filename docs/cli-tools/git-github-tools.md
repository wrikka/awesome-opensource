# Git & GitHub Tools

| Reference | CLI | Description | Benefit |
|-----------|-----|-------------|---------|
| [act](https://github.com/nektos/act) | `act` | Run GitHub Actions locally | Test GitHub workflows before pushing |
| [gh](https://cli.github.com/) | `gh` | GitHub CLI | Manage GitHub from command line |
| [git](https://git-scm.com/) | `git` | Version control | Version control system |
| [gitleaks](https://github.com/gitleaks/gitleaks) | `gitleaks` | Secret scanner | Scan secrets in code |
| [gitui](https://github.com/extrawurst/gitui) | `gitui` | Git TUI | Manage Git with terminal UI |
| [git-split-diffs](https://github.com/banga/git-split-diffs) | `git-split-diffs` | Git diff viewer | View git diffs beautifully |
| [lazygit](https://github.com/jesseduffield/lazygit) | `lazygit` | Git TUI | Beautiful and easy Git UI |
| [lefthook](https://github.com/evilmartians/lefthook) | `lefthook` | Git hooks manager | Config-driven Git hooks automation |
| [git-x](https://github.com/MichaelAquilina/git-x) | `git-x` | Git extensions | Additional Git commands |
