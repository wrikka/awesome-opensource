# File Operations

| Reference | CLI | Description | Benefit |
|-----------|-----|-------------|---------|
| [eza](https://github.com/eza-community/eza) | `eza` | Modern ls replacement | Modern ls with beautiful colors and full info |
| [bat](https://github.com/sharkdp/bat) | `bat` | Cat clone with syntax highlighting | Display files with beautiful syntax highlighting |
| [broot](https://github.com/Canop/broot) | `broot` | Terminal file explorer | Fast and powerful file browsing in terminal |
| [yazi](https://yazi-rs.github.io/) | `yazi` | Terminal file manager | Modern and fast file manager |
| [dua](https://github.com/Byron/dua-cli) | `dua` | Disk usage analyzer | Interactive disk space analysis |
| [duf](https://github.com/muesli/duf) | `duf` | Disk usage free | Beautiful view of used and free disk space |
| [file](https://darwinsys.com/file/) | `file` | File type detection | Check file types |
