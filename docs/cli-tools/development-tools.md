# Development Tools

| Reference | CLI | Description | Benefit |
|-----------|-----|-------------|---------|
| [biome](https://biomejs.dev/) | `biome` | Linter and formatter | Faster lint and format than ESLint/Prettier |
| [dprint](https://dprint.dev/) | `dprint` | Code formatter | Multi-language formatter with pluggable architecture |
| [oxlint](https://oxc.rs/) | `oxlint` | Fast linter | Fast JavaScript/TypeScript linter in Rust |
| [rust-analyzer](https://rust-analyzer.github.io/) | `rust-analyzer` | Rust LSP | Language Server for Rust |
