# Awesome Open Source Tools

Collection of CLI and TUI tools installed on the machine with their commands and benefits.

This repository consolidates tools from both CLI and TUI categories, providing a comprehensive reference for open source tools used in development.

## Quick Start

- [CLI Tools](/cli-tools/archive-tools) - Command-line interface tools
- [TUI Tools](/tui-tools/system-monitoring) - Terminal user interface tools
