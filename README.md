# Awesome Open Source Tools

Collection of CLI and TUI tools installed on the machine with their commands and benefits

This repository consolidates tools from both CLI and TUI categories, providing a comprehensive reference for open source tools used in development.

## Documentation

For detailed documentation, visit the [docs site](https://wrikka.github.io/awesome-opensource/)

## Quick Reference

### CLI Tools

- **Archive Tools**: Compression and extraction utilities
- **Git & GitHub Tools**: Version control and GitHub automation
- **Code Analysis & Search**: Code navigation and analysis
- **File Operations**: File management and disk usage
- **Development Tools**: Linters, formatters, and language servers
- **Package Managers**: Runtime and dependency management
- **Runtime & Languages**: Programming language runtimes
- **Shell & Terminal**: Shell enhancements and terminal emulators
- **Text Editors**: Terminal-based text editors
- **System Monitoring**: Resource monitoring and system info
- **Benchmarking**: Performance measurement
- **Containerization**: Container orchestration
- **Configuration Management**: Dotfile and secrets management
- **Code Counting**: Line counting and complexity analysis
- **Build Tools**: Compilation and build automation
- **Network Tools**: Network utilities
- **Media Processing**: Video and audio processing
- **Task Management**: Task and todo management
- **Misc Tools**: Various utilities

### TUI Tools

- **System Monitoring**: Interactive resource monitoring
- **File Management**: Terminal file explorers
- **Git Tools**: Git terminal interfaces
- **Text Editors**: Modal and simple terminal editors
- **Terminal Emulators**: Modern terminal emulators
- **Task Management**: Task management interfaces
- **Kubernetes**: Kubernetes management interfaces
- **Misc**: Other TUI utilities
